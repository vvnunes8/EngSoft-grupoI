Descrição Informal

Hoje em dia, com a vida agitada, a gestão de compras e estoque pode se tornar uma tarefa difícil e cansativa. Muitas vezes, as pessoas enfrentam o problema de esquecer itens essenciais ao fazer suas compras no mercado, o que a obriga a retornar no estabelecimento, perdendo um tempo precioso com filas e, em alguns casos, trânsito.

Para solucionar esse caso, propõe-se criação de um sistema inteligente de acompanhamento de estoque de comidas, que não apenas auxilia na gestão de compras, mas também sugere receitas baseadas nos ingredientes disponíveis.

Valor Entregue

O sistema visa PREVENIR O ESFORÇO COM TRANSPORTE E OTIMIZAR O TEMPO DA ROTINA do usuário que antes sofria com uma rotina repleta de compromissos. Além de propor um consumo mais ECONÔMICO E SUSTENTÁVEL.

Histórias do Usuário

Casos de Uso

  1. Como dono(a) de casa eu quero criar uma lista de compras do mês para minha casa para que o software gere uma lista adequada ao meu perfil para acessar mensalmente visando qualidade do produto e evitar que eu esqueça algo.
  2. Como dono do mercado eu quero catalogar os preços e produtos no software para que seja possível otimizar a experiência do cliente nos meus estabelecimentos.
  3. Como dono de lanchonetes eu quero uma lista de compras personalizadas para que seja possível comprar em quantidade com o melhor custo benefício.
  4. Como dono de um empreendimento alimentício eu quero o nome da minha empresa exposto neste software para fins de marketing/receita.
  5. Como dono(a) de casa eu quero ter acesso às ofertas dos mercados próximos a mim para comparar custo e benefício de acordo com minhas necessidades.
  6. Como dono(a) de lanchonetes eu quero poder acompanhar o histórico de preço do produto para efetuar minhas compras no melhor momento possível.
  7. Como dono(a) de casa eu quero poder acompanhar o histórico de preço do produto para efetuar minhas compras no melhor momento possível.

Caso de Uso 1: Acessar Lista de Compras Personalizada

Ator principal:

Dono(a) de Casa

Pré-condições:

Dono(a) de Casa está logado ao sistema

Gatilho:

Dono(a) de Casa escolhe acessar Listas de Compras Personalizada

Cenário Principal:

  1. Sistema pede para o(a) dono(a) de casa confirmar o pedido.
  2. Dono(a) de casa confirma o pedido.
  3. Sistema verifica se o dono(a) de casa tem compras registradas.
  4. Sistema gera a lista de compras personalizada.
  5. Sistema informa que a lista de compras foi gerada.
  6. Sistema coloca em tela a lista de compras personalizada.

Cenário Alternativo:

  1. 3.a Dono(a) de casa não tem compras registradas no sistema:
  2. Sistema informa para o(a) dono(a) de casa que não é possível gerar a lista de compras personalizada.
  3. Dono(a) de casa aceita a mensagem.

Caso de Uso 2: Catalogar Preços e Produtos

Ator principal:

Dono de Mercado

Pré-condições:

Dono(a) de Mercado está logado ao sistema

Gatilho:

Dono de Mercado escolhe catalogar preços e produtos

Cenário Principal:

  1. Sistema pede para Dono de Mercado confirmar o pedido.
  2. Sistema abre a funcionalidade de cadastro de produtos.
  3. Dono do Mercado insere nome, descrição, categoria e preço do produto.
  4. Sistema confirma o registro do novo produto.
  5. Sistema adiciona novo produto ao catálogo do mercado.

Cenário Alternativo:

  1. 3.a Dono do Mercado insere um produto com informações incompletas:
  2. Sistema informa sobre campos obrigatórios não preenchidos.
  3. Dono do Mercado cadastra um produto com as informações completas.
  4. Sistema confirma o registro do produto.

Caso de Uso 3: Incluir publicidade no aplicativo

Ator principal:

Anunciante

Pré-condições:

Anunciante está logado no sistema

Gatilho:

Anunciante acessa as configurações de marketing

Cenário Principal:

  1. Sistema abre as configurações de marketing.
  2. Anunciante insere o nome do estabelecimento.
  3. Sistema confirma a inclusão do nome do estabelecimento.

Cenário Alternativo:

  1. 2.a Anunciante insere um nome muito longo:
  2. Sistema alerta sobre o tamanho máximo permitido para o nome.
  3. Anunciante ajusta o nome dentro dos limites.
